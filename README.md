# LightCraft Mod
A lightweight Fabric mod for 1.21.1.

## Installation
1. Install Fabric Loader.
2. Run `./gradlew build` (Linux/Mac) or `gradlew build` (Windows).
3. Check `build/libs` for the JAR.

## Features
- FPS Counter (Press H)
- Minimap (Press M)
- Waypoints (Press B)
- Config (Press K)
